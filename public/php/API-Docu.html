War API Documentation


The API returns all wars, which ever exists and is regularly updated from the crest endpoint. The API provides the WarID, both the Aggressor and Defender Name, ID and GroupTypeID (2 = corp, 32= alliance) and an status ( 0=declared,1=active,2=finished).


The API returns 200 entries per request.


The following GET-Parameters you can use:

AgrID = ID of the Aggressor

DefID = ID of the Defender

ID = ID of the Aggressor or Defender

status = status of the war

page = the page of the api (starts with 1)


You can use AgrID and DefID or ID not both combinations at the same time

You can add multiple value to a parameter using a „+“.


API-URL: https://www.warcontacts.tk/php/api.php


Example-URL: https://www.warcontacts.tk/php/api.php?ID=144740602&status=2&page=1
